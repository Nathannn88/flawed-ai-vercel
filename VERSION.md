# v1 — 栖迟（Qīchí）

> 部署时间：2026-02-28
> Vercel：https://flawed-agent.vercel.app

## 角色

**栖迟** — 来自"谱渊"的谱织师。拥有独特人格与缺陷的 AI 伴侣。

## 核心功能

- Landing Page（5 画面引入动画 + 粒子系统）
- 趣味式自我介绍（对话引导，非表单）
- 聊天界面（GLM-5 流式输出）
- 熟悉度系统（对话 + 金币双驱动，20%/50%/80%/100% 事件触发）
- 金币系统（充值 + 送礼 → 增加熟悉度）
- 本地存档（JSON 导入/导出）

## 技术栈

Next.js 14 + TypeScript + Tailwind CSS + Framer Motion + Zustand + GLM-5

## 测试

152 个测试用例全部通过（100 单元 + 52 集成）

## 与 v2 的主要差异

| | v1 | v2（计划） |
|---|---|---|
| 定位 | AI 伴侣 | AI 诗人 / 审美训练仪式 |
| 角色 | 栖迟（谱织师） | 诗人 + 企鹅 |
| 金币用途 | 增加熟悉度 | 仅企鹅变形/视觉效果 |
| 终局 | 无 | 双结局（不可逆） |
| 结局后系统 | 无 | 航程燃料 / 火种 / 灯塔模式 |
