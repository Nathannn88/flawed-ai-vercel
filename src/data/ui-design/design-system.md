### 失速灰化色板

失速状态覆盖所有阶段色彩——通过 CSS filter 叠加和变量覆盖实现。

| 状态 | filter | 附加效果 |
|------|--------|---------|
| **正常** | `saturate(1) brightness(1)` | 无 |
| **预警**（燃料<30） | `saturate(0.8) brightness(0.95)` | 边缘 `box-shadow: inset 0 0 60px rgba(90,90,90,0.15)` |
| **失速**（燃料<15） | `saturate(0.5) brightness(0.85)` | 边缘 `box-shadow: inset 0 0 100px rgba(60,60,60,0.3)` + 灰色蒙层 `rgba(40,40,50,0.25)` |
| **深度失速**（燃料=0） | `saturate(0.15) brightness(0.7)` | 叠加静态噪声纹理（SVG `feTurbulence`）+ `rgba(30,30,40,0.5)` 蒙层 |

失速灰化覆盖变量：

```css
[data-stall="warning"] {
  --stall-sat: 0.8;
  --stall-bright: 0.95;
  --stall-overlay: rgba(90, 90, 90, 0.15);
}
[data-stall="stalled"] {
  --stall-sat: 0.5;
  --stall-bright: 0.85;
  --stall-overlay: rgba(60, 60, 60, 0.3);
}
[data-stall="deep"] {
  --stall-sat: 0.15;
  --stall-bright: 0.7;
  --stall-overlay: rgba(30, 30, 40, 0.5);
}
```

## 二、字体方案

### 字体族

| 角色 | 字体名 | 备选 | CSS 变量 | 用途 |
|------|--------|------|----------|------|
| 中文正文 | **Noto Sans SC** | "PingFang SC", "Microsoft YaHei" | `--font-cn` | 所有中文正文、对话、UI 文字 |
| 英文标题 | **Space Grotesk** | "Outfit", system-ui | `--font-display` | Landing 大标题、品牌文字、按钮 |
| 诗人引用/火种 | **Crimson Pro** italic | "Noto Serif SC", Georgia | `--font-poetic` | 诗人特殊引用、火种卡片、事件台词 |
| 等宽字体 | **JetBrains Mono** | "Fira Code", monospace | `--font-mono` | 远方消息、系统信息、调试 |

设计意图：
- **Noto Sans SC**：清朗、现代但不冰冷，中文阅读体验优秀，与诗意内容不冲突
- **Space Grotesk**：几何无衬线但带有手工感的墨迹终端，比 Outfit 更有性格
- **Crimson Pro italic**：经典衬线体斜体，优雅但不矫揉，用于诗人最重要的句子
- **JetBrains Mono**：等宽字体中最具辨识度的，用于"远方世界"的通信——暗示信号/数据传输

## 四、动画规范

### 缓动曲线

| 名称 | CSS 值 | 用途 |
|------|--------|------|
| `ease-default` | `cubic-bezier(0.4, 0, 0.2, 1)` | 通用过渡 |
| `ease-enter` | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入 |
| `ease-exit` | `cubic-bezier(0.4, 0, 1, 1)` | 元素退出 |
| `ease-bounce` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 弹性效果（企鹅弹跳） |
| `ease-breath` | `cubic-bezier(0.4, 0, 0.6, 1)` | 呼吸脉动 |
| `ease-dramatic` | `cubic-bezier(0.16, 1, 0.3, 1)` | 戏剧性入场（事件演出） |
| `ease-gentle` | `cubic-bezier(0.25, 0.1, 0.25, 1)` | 缓慢自然（灯塔光束） |

### 持续时间标准

| 名称 | 时长 | 用途 |
|------|------|------|
| `fast` | 150ms | 按钮状态、hover 变色 |
| `normal` | 250ms | 标准过渡、面板展开 |
| `slow` | 400ms | 气泡出现、阶段切换 |
| `dramatic` | 800ms | 事件入场、重要状态变化 |
| `cinematic` | 1500ms | 结局帧转场、灯塔光束 |
| `epic` | 3000ms+ | 变形动画、结局演出序列 |

### 企鹅动画

#### 基础微动画（idle）

| 动画 | 参数 | 说明 |
|------|------|------|
| 谱纹呼吸 | 3s, ease-breath, infinite | 腹部翡翠绿谱纹 opacity 在 0.3-0.8 间脉动 |
| 边缘抖动 | 随机 0.05-0.15s, steps(2) | SVG 描边的微小偏移，模拟信号干扰 |
| 双眼亮度 | 4s, ease-breath, infinite | 左眼翡翠绿、右眼琥珀金的亮度微变 |
| 身体摇晃 | 8s, ease-gentle, infinite | ±2度的极微旋转 |

#### 对话活跃态

| 触发 | 变化 |
|------|------|
| 用户输入中 | 双眼亮度 +30%，边缘抖动频率 ×2 |
| 消息发送 | 0.3s 弹跳 (ease-bounce)，谱纹闪亮一次 |
| 诗人回复中 | 谱纹流动速度 ×3，模拟"思考" |
| 长时间沉默 | 边缘抖动停止，双眼暗至 40%，偶尔"叹息"（0.5s 收缩） |

#### 失速态动画

| 状态 | 变化 |
|------|------|
| 预警 | 所有动画速度 ×0.7，偶尔打寒颤（0.2s 快速抖动） |
| 失速 | 除微弱呼吸外全部停止，呼吸周期延长至 6s |
| 深度失速 | 完全静止，仅保留 opacity 0.4 的灰色轮廓 |

### 变形动画时间线

总时长 **2.4 秒**，24fps 关键帧。

```
0.0s ─ 频率解构开始
       边缘线条分裂为波形
       翡翠绿同心圆波纹扩散
0.4s ─ 轮廓 50% 解构
0.8s ─ 完全解构为目标主色光球
       表面闪过目标形态局部剪影
1.2s ─ 光球脉动达到峰值
1.6s ─ 频率凝结开始
       从核心向外凝固
1.8s ─ 双眼最先出现
2.0s ─ 轮廓 80% 凝固
2.4s ─ 白色光线沿轮廓扫一圈，完成
```

退回默认：1.2s，颜色一层层剥离（像脱壳）。

### 失速灰化过渡

| 过渡 | 时长 | 缓动 |
|------|------|------|
| 正常 → 预警 | 3s | ease-default |
| 预警 → 失速 | 5s | ease-gentle |
| 失速 → 深度失速 | 8s | ease-gentle |
| 任意恢复（上升） | 2s | ease-dramatic |

```
颜色：linear-gradient(to bottom, var(--chord-color-1), var(--chord-color-2))
透明度：var(--chord-opacity)
动画：spectrum-flow 8s linear infinite（颜色沿竖直方向流动）
数量：3-5 条，间距不等，随机偏移
```

随阶段变化——颜色、透明度、流动速度均通过 CSS 变量控制。

### 环境粒子

背景中极微弱的光点，增强深空感。

```
颜色：var(--particle-color)
尺寸：1-3px
透明度：0.15-0.4
数量：20-40 个
运动：随机缓慢飘浮，速度受 var(--particle-speed) 控制
```

Canvas 或 CSS animation 实现，不影响交互层。

### 噪点纹理

用于远方消息气泡和深度失速状态。

```
SVG feTurbulence：
  type="fractalNoise"
  baseFrequency="0.65"
  numOctaves="3"
  opacity: 0.04（远方消息）/ 0.08（深度失速）
```

## 八、无障碍与性能

### 无障碍

- 所有颜色组合确保 WCAG AA 对比度（4.5:1 正文，3:1 大文字）
- focus-visible 轮廓统一为 2px solid var(--phase-primary)
- prefers-reduced-motion 时禁用所有动画
- 所有按钮/交互元素有 aria-label
- 企鹅区域有 role="img" + alt 描述

### 性能

- 动画优先使用 transform 和 opacity（GPU 加速）
- 粒子系统使用 Canvas（不用 DOM 元素）
- 毛玻璃效果限制嵌套层数（最多 2 层）
- 大型动画序列（结局）使用 requestAnimationFrame + 分帧加载
- 灯塔海面使用 CSS animation，不用 JS
